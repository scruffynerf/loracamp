# sample.toml Reference

Samples are the outputs generated by a Model, such as audio demos or preview images. Each sample can have its own `sample.toml` manifest to provide individual metadata.

## Setup

Place a `sample.toml` alongside your sample file. If the sample file is `my_demo.mp3`, LoraCamp will look for `my_demo.sample.toml` or just `sample.toml` in a dedicated sample directory.

```text
cinematic-style/
├─ model.toml
├─ model.safetensors
└─ samples/
   ├─ demo_1.mp3
   └─ demo_1.sample.toml   <--- Sample Manifest
```

## Options

### `title`

The display name of the sample. If omitted, the filename is used.

```toml
title = "Moody Night Rain Demo"
```

### `creator` (String) | `creators` (List)

The creator(s) of this specific sample (if different from the main Model creator).

```toml
creator = "SampleArtisan"
```

### `tags` (Optional)

A list of classification tags for this specific sample. These are used for search indexing.

```toml
tags = ["jazz", "upbeat", "demo"]
```

### `sample_number` (Optional)

The index of the sample (e.g. `1`). Used for sorting.

Information about how this sample was created, including prompts, settings, or seeds used.

```toml
about = "Generated using the 'noir' trigger word with a guidance scale of 7.5."
```

### `prompt` (Optional)

The specific prompt used to generate this sample.

```toml
prompt = "a lonely street lamp in the fog, film noir style"
```

### `negative_prompt` (Optional)

The negative prompt used during generation.

```toml
negative_prompt = "sunshine, happy, colorful"
```

### `seed` (Optional)

The random seed used for generation.

```toml
seed = 123456789
```

### `model` (Optional)

The base checkpoint or model used to generate this sample.

```toml
model = "LargeCheckpoint-v2"
```

### `vae` (Optional)

The VAE (Variational Auto Encoder) used.

```toml
vae = "vae-ft-mse-840000-ema-pruned.safetensors"
```

### `clip` (Optional)

The CLIP (Text Encoder) or CLIP skip settings.

```toml
clip = "clip-l"
```

### `loras` (Optional)

A list of additional LoRAs used in this generation, with optional weights.

```toml
[[loras]]
name = "DetailSlider"
weight = 0.5

[[loras]]
name = "StyleTransfer"
weight = 1.0
```

### `steps` (Optional)

Number of sampling steps.

```toml
steps = 25
```

### `cfg` (Optional)

The CFG (Classifier Free Guidance) scale.

```toml
cfg = 7.5
```

### `sampler` (Optional)

The sampler used (e.g., Euler a, DPM++ 2M).

```toml
sampler = "Euler a"
```

### `scheduler` (Optional)

The noise scheduler (e.g., Karras, Exponential).

```toml
scheduler = "Karras"
```

### `workflow` (Optional)

A path to a local workflow file (e.g., ComfyUI JSON) or a URL to a hosted workflow.

```toml
workflow = "workflows/my_comfy_setup.json"

# OR

workflow = "https://openart.ai/workflows/..."
```

### `shift` (Optional)

Pitch or time shift applied.

```toml
shift = 1.2
```

### `keyscale` (Optional)

The musical key or scale (e.g., "C Major", "D Minor").

```toml
keyscale = "A# Phrygian"
```

### `bpm` (Optional)

The tempo in Beats Per Minute.

```toml
bpm = 120.0
```

### `timesignature` (Optional)

The musical time signature (e.g., "4/4", "3/4").

```toml
timesignature = "7/8"
```

### `language` (Optional)

The language of the content (e.g., "en", "ko").

```toml
language = "jp"
```

### `additional_seed` (Optional)

A secondary seed used in multi-seed generation processes.

```toml
additional_seed = 987654321
```

### `llm_cfg` (Optional)

The guidance scale for Large Language Model generation.

```toml
llm_cfg = 1.1
```

### `llm_temp` (Optional)

The temperature for LLM sampling (controls randomness).

```toml
llm_temp = 0.8
```

### `llm_top_p` (Optional)

Top-p (nucleus) sampling threshold for LLMs.

```toml
llm_top_p = 0.95
```

### `llm_top_k` (Optional)

Top-k sampling threshold for LLMs.

```toml
llm_top_k = 50
```

### `llm_min_p` (Optional)

Min-p sampling threshold for LLMs.

```toml
llm_min_p = 0.05
```

### `llm_rep_penalty` (Optional)

Repetition penalty for LLM text generation.

```toml
llm_rep_penalty = 1.2
```

### `cover` (Optional)

The filename of a cover image for this specific sample. If omitted, LoraCamp looks for an image with the same stem as the audio file (e.g., `demo_1.jpg` for `demo_1.mp3`), then falls back to `preview.*` or `cover.*` in the same directory, and finally falls back to the model's main preview image.

```toml
cover = "demo_1_cover.png"
```

### `links` (Optional)

A list of external links for this sample.

```toml
[[links]]
label = "Listen on SoundCloud"
url = "https://soundcloud.com/..."
```

### `external_page` (Optional)

A URL to link the sample title to. Useful for linking to an external page with more details about this specific sample.

```toml
external_page = "https://example.com/my-demo-details"
```

### `copy_link` (Optional)

Whether to show a "Copy Link" button for this sample. Defaults to `true`.

```toml
copy_link = false
```

### `sample_extras` (Optional)

A list of additional files specific to this sample (e.g. a high-pass version).

### `opengraph` (Optional)

Provide page-specific overrides if this sample is ever displayed on its own page (currently inherits from Model).

### `lyrics` (Optional)

The lyrics or transcript associated with the sample. TOML supports multi-line strings using triple quotes (`"""`), which is ideal for long text like lyrics.

```toml
lyrics = """
[Chorus]
Oh, the rain in Spain
Stays mainly in the plain!

[Verse 1]
In Hartford, Hereford, and Hampshire,
Hurricanes hardly ever happen.
"""
```
