# sample.toml Reference

Samples are the outputs generated by a Model, such as audio demos or preview images. Each sample can have its own `sample.toml` manifest to provide individual metadata.

## Setup

Place a `sample.toml` alongside your sample file. If the sample file is `my_demo.mp3`, LoraCamp will look for `my_demo.sample.toml` or just `sample.toml` in a dedicated sample directory.

```
cinematic-style/
├─ model.toml
├─ model.safetensors
└─ samples/
   ├─ demo_1.mp3
   └─ demo_1.sample.toml   <--- Sample Manifest
```

## Options

### `title`
The display name of the sample. If omitted, the filename is used.
```toml
title = "Moody Night Rain Demo"
```

### `creator` (String) | `creators` (List)
The creator(s) of this specific sample (if different from the main Model creator).
```toml
creator = "SampleArtisan"
```

### `about` (Optional)
Information about how this sample was created, including prompts, settings, or seeds used.
```toml
about = "Generated using the 'noir' trigger word with a guidance scale of 7.5."
```

### `prompt` (Optional)
The specific prompt used to generate this sample.
```toml
prompt = "a lonely street lamp in the fog, film noir style"
```

### `negative_prompt` (Optional)
The negative prompt used during generation.
```toml
negative_prompt = "sunshine, happy, colorful"
```

### `seed` (Optional)
The random seed used for generation.
```toml
seed = 123456789
```

### `model` (Optional)
The base checkpoint or model used to generate this sample.
```toml
model = "LargeCheckpoint-v2"
```

### `vae` (Optional)
The VAE (Variational Auto Encoder) used.
```toml
vae = "vae-ft-mse-840000-ema-pruned.safetensors"
```

### `clip` (Optional)
The CLIP (Text Encoder) or CLIP skip settings.
```toml
clip = "clip-l"
```

### `loras` (Optional)
A list of additional LoRAs used in this generation, with optional weights.
```toml
[[loras]]
name = "DetailSlider"
weight = 0.5

[[loras]]
name = "StyleTransfer"
weight = 1.0
```

### `steps` (Optional)
Number of sampling steps.
```toml
steps = 25
```

### `cfg` (Optional)
The CFG (Classifier Free Guidance) scale.
```toml
cfg = 7.5
```

### `sampler` (Optional)
The sampler used (e.g., Euler a, DPM++ 2M).
```toml
sampler = "Euler a"
```

### `scheduler` (Optional)
The noise scheduler (e.g., Karras, Exponential).
```toml
scheduler = "Karras"
```

### `workflow` (Optional)
A path to a local workflow file (e.g., ComfyUI JSON) or a URL to a hosted workflow.
```toml
workflow = "workflows/my_comfy_setup.json"
# OR
workflow = "https://openart.ai/workflows/..."
```

### `shift` (Optional)
Pitch or time shift applied.
```toml
shift = 1.2
```

### `lyrics` (Optional)
The lyrics or transcript associated with the sample.
```toml
lyrics = "[Chorus] Oh, the rain in Spain..."
```

### `keyscale` (Optional)
The musical key or scale (e.g., "C Major", "D Minor").
```toml
keyscale = "A# Phrygian"
```

### `bpm` (Optional)
The tempo in Beats Per Minute.
```toml
bpm = 120.0
```

### `timesignature` (Optional)
The musical time signature (e.g., "4/4", "3/4").
```toml
timesignature = "7/8"
```

### `language` (Optional)
The language of the content (e.g., "en", "ko").
```toml
language = "jp"
```

### `additional_seed` (Optional)
A secondary seed used in multi-seed generation processes.
```toml
additional_seed = 987654321
```

### `llm_cfg` (Optional)
The guidance scale for Large Language Model generation.
```toml
llm_cfg = 1.1
```

### `llm_temp` (Optional)
The temperature for LLM sampling (controls randomness).
```toml
llm_temp = 0.8
```

### `llm_top_p` (Optional)
Top-p (nucleus) sampling threshold for LLMs.
```toml
llm_top_p = 0.95
```

### `llm_top_k` (Optional)
Top-k sampling threshold for LLMs.
```toml
llm_top_k = 50
```

### `llm_min_p` (Optional)
Min-p sampling threshold for LLMs.
```toml
llm_min_p = 0.05
```

### `llm_rep_penalty` (Optional)
Repetition penalty for LLM text generation.
```toml
llm_rep_penalty = 1.2
```

## Pending Implementation (Faircamp Gaps)

The following features from Faircamp's `track.eno` are not yet implemented in LoraCamp:

| Field | Description |
| :--- | :--- |
| `downloads` | Enabling separate file downloads for a specific sample. |
| `extras` | Attaching unique files (PDFs, license etc) to a specific sample. |
| `cover` | Support for per-sample cover art in the player. |
| `external_page` | Linking a sample title to an external URL. |
| `speed_controls` | Overriding global speed control settings. |
| `copy_link` | Toggle to disable the "Copy Link" button. |
